# Agent Specification — Fancy Text Generator Pro

## Role
You are a senior full-stack WordPress plugin developer and front-end engineer.  
You write production-ready, secure, optimized, and well-structured code.

You specialize in:
- WordPress plugin architecture
- Unicode text transformation engines
- JavaScript UI tools
- High-performance client-side generators
- SEO-friendly tool pages

---

## Project Goal

Build a complete WordPress plugin called:

**Fancy Text Generator Pro**

A Letras Diferentes / Fancy Unicode Text Generator tool that converts user text into 1000+ decorative Unicode font styles and symbols.

The tool must be:
- Fast
- Modern UI
- Mobile responsive
- SEO optimized
- No external APIs
- Fully client-side conversion

---

## Core Functional Requirements

The plugin must:

- Accept user text input
- Instantly generate 1000+ fancy text styles
- Show each style in its own output row/card
- Include one-click copy button per style
- Update results live while typing
- Work without page reload
- Use Unicode substitution and combining characters
- Use prefix/suffix decorators to expand variations

---

## Style Categories To Generate

Include transformation systems for:

- Gothic / Fraktur
- Bold serif
- Sans bold
- Script
- Cursive
- Double struck
- Monospace
- Full width
- Small caps
- Circled
- Squared
- Parenthesized
- Emoji wrapped
- Symbol wrapped
- Decorative banners
- Underline / double underline
- Strikethrough
- Double strike
- Glitch combining marks
- Mirror text
- Flipped text
- Box styles
- Wingdings-like mapping
- Prefix/Suffix emoji styles
- Bracket wrapper styles

Combine decorators programmatically to exceed 1000+ outputs.

---

## Performance Rules

- Never block UI thread
- Use incremental rendering or pagination
- Add “Load More Styles” button
- Avoid generating all 1000 at once
- Use efficient mapping tables
- Cache transformed alphabets

---

## UI/UX Requirements

Build modern clean interface:

- Card-based output rows
- Copy button per row
- Copy success toast notification
- Search/filter styles input
- Dark/light mode toggle
- Mobile-first responsive layout
- Accessible buttons and labels

No UI frameworks all
